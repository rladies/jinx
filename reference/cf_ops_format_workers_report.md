# Format Workers invocation metrics as a markdown table

Format Workers invocation metrics as a markdown table

## Usage

``` r
cf_ops_format_workers_report(df)
```

## Arguments

- df:

  Data frame from
  [`cf_ops_workers_invocations()`](https://rladies.github.io/jinx/reference/cf_ops_workers_invocations.md).

## Value

Character string with markdown-formatted report.
