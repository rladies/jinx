# Export the locked slot of a poll as an iCalendar (.ics) string

Export the locked slot of a poll as an iCalendar (.ics) string

## Usage

``` r
meeting_poll_ics(id, base_url = samkoma_base_url())
```

## Arguments

- id:

  Poll id.

- base_url:

  API base URL.

## Value

The .ics file contents as a character string.
