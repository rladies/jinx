# Compute activity trends

Calculates month-over-month changes and rolling averages from chapter
activity data.

## Usage

``` r
compute_activity_trends(activity_data)
```

## Arguments

- activity_data:

  Data frame from
  [`collect_chapter_activity()`](https://rladies.github.io/jinx/reference/collect_chapter_activity.md).

## Value

Data frame with trend columns added.
