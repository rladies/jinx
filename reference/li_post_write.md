# Post to LinkedIn

Post to LinkedIn

## Usage

``` r
li_post_write(author, text, image = NULL, image_alt = "")
```

## Arguments

- author:

  LinkedIn URN of the author.

- text:

  Post text.

- image:

  Optional file path to an image.

- image_alt:

  Alt text for the image.

## Value

LinkedIn post ID (invisibly).
