# Compute activity trends

Calculates month-over-month changes and rolling averages from chapter
activity data.

## Usage

``` r
analytics_compute_trends(activity_data)
```

## Arguments

- activity_data:

  Data frame from
  [`analytics_collect_chapter_activity()`](https://rladies.github.io/jinx/reference/analytics_collect_chapter_activity.md).

## Value

Data frame with trend columns added.
