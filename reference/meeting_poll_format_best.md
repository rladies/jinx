# Format ranked best slots as markdown

Format ranked best slots as markdown

## Usage

``` r
meeting_poll_format_best(best, title = NULL, top = 3)
```

## Arguments

- best:

  A data frame from
  [`meeting_poll_best()`](https://rladies.github.io/jinx/reference/meeting_poll_best.md).

- title:

  Optional poll title for the heading.

- top:

  Maximum number of slots to show.

## Value

Character string of markdown.
